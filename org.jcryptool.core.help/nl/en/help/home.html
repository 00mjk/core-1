JCrypTool Help

Welcome to the JCrypTool online help! The extensive help system for the JCrypTool e-learning platform consists of three categories:

  • The User Guide describes all functions JCrypTool offers; from the various editors up to the plug-ins for encrypting, signing and many more. This is an optimal entry point if you have never used JCrypTool before.
  • The Developer Guide contains information for interested developers and advanced users who wish to extend JCrypTool with their own plug-ins.
  • Cryptology contains information regarding the theory of cryptography, algorithms, digital signatures, encryption, and more.

As with the rest of JCrypTool the online help is extremely extensible. And as with the plug-ins itself, every crypto plug-in is responsible for contributing its own help content to the User Guide. So if you can't find any help for a certain plug-in the developer most likely didn't write one yet.

More information about JCrypTool is available on our home page. The source code for both JCrypTool Core and JCrypTool Crypto is available for download as well.
