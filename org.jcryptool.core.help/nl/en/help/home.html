JCrypTool Help

Welcome to the JCrypTool (JCT) online help!

The User Guide contains an introduction to JCT. It describes the functions JCT offers and how to use them: From the various editors and the visualizations to the plug-ins for encryption, digital signatures, etc.
This is the best entry point if you have never used JCT before.

To open the User Guide click on User Guide in the table of contents on the left side. Now you can see the different categories in which the help is categorized in the JCT, e.g. General Information or Cryptology. You can open these by clicking on the respective entry. This way you can navigate through to the help page you are looking for.

Alternatively, you can use the search box at the top left of the page to get help on a specific topic. The best way to do this is to search for the name of a plugin or procedure.

Developers find relevant and up-to-date information in our GitHub Wiki.

More information about JCT is available on our home page. The source code for JCrypTool is freely available on GitHub.
