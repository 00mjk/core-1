JCrypTool Help

Welcome to the JCrypTool online help!

From now on we use the abbreviation ‘JCT’ for JCrypTool.

The help system for the JCrypTool e-learning platform consists of two parts:

  • The User Guide contains an introduction to JCT. It describes the functions JCT offers and how to use them: From the various editors and the visualizations to the plug-ins for encryption, digital signatures, etc.
    This is the best entry point if you have never used JCT before.
  • The Developer Guide contains information for interested developers and advanced users who wish to extend JCT with their own plug-ins.

As with the rest of JCrypTool the online help is modularly extensible. Every crypto plug-in is responsible for delivering its own help content to the User Guide. If you can't find any help for a certain plug-in please inform us (we then contact the author of the plug-in).

More information about JCT is available on our home page. The source code for both, JCrypTool Core and JCrypTool Crypto, is available freely for download.
