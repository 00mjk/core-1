JCrypTool Help

Welcome to the JCrypTool online help! The help system for the JCrypTool e-learning platform consists of two categories:

  • The User Guide describes the functions JCrypTool offers: From the various editors and the visualisations up to the plug-ins for encrypting, signing, etc.
    This is the best entry point if you have never used JCrypTool before.
  • The Developer Guide contains information for interested developers and advanced users who wish to extend JCrypTool with their own plug-ins.

As with the rest of JCrypTool the online help is modularly extensible. Every crypto plug-in is responsible for delivering its own help content to the User Guide. So if you can't find any help for a certain plug-in the developer most likely didn't write one yet.

More information about JCrypTool is available on our home page. The source code for both, JCrypTool Core and JCrypTool Crypto, is available freely for download.
