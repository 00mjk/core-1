Algorithms

The JCT main menu Algorithms (in the Default perspective) has the following entries:

Asymmetric Asymmetric methods for encryption (and signatures)
Classic Classic cipher methods
Hash Hash methods
MAC Message authentication codes
Random Number Generator Pseudo random number generator (PRNG)
Signature Signature methods
Symmetric Modern symmetric ciphers
Keystore JCT keystore (from the Algorithm perspective)


Remark:

The algorithms in the JCT main menu Algorithms (in the Default perspective) are just a few selected cryptographic algorithms to be executed directly via a dialog box.

Many more cryptographic algorithms in JCT can be found at

  • Visuals main menu – Visualizations of cryptographic methods
  • Algorithm perspective – Offers the algorithms from the crypto providers Bouncy Castle and FlexiProvider (only modern algorithms)

A tabular overview about all cryptographic functions in all CrypTool programs including JCT can be found at the CrypTool website in the function overview. Having there the check set for JCT only, shows that JCT contains more than 100 function groups and more than 200 single functions (paths).
