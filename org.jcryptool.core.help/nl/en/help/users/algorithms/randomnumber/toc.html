Random number generation

Random number generation (RNG) is a process which generates a sequence of numbers or symbols that cannot be reasonably predicted better than by a random chance. Random number generators can be either true hardware random-number generators (HRNGs), which generate random numbers as a function of current values of some physical environment attribute that is constantly changing in a manner that is practically impossible to model, or pseudo-random number generators (PRNGs), which generate numbers that look random, but are actually deterministic, and can be reproduced if the state of the PRNG is known.

Several computational methods for pseudo-random number generation exist. The generation of pseudo-random numbers is an important and common task in computer programming. The design of cryptographically secure pseudorandom number generators (CSPRNG) or cryptographic pseudorandom number generators (CPRNG) can be based on cryptographic primitives such as ciphers and cryptographic hashes (e.g. SHA-1, SHA-2, SHA-3).
