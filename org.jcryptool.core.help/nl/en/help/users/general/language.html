Changing the Language

The default JCrypTool language is based on your operating systems regional settings. An English OS normally launches an English JCrypTool.

In order to change the JCrypTool language, open the Preferences and switch to the General page. Simply select the desired language. JCrypTool must be restarted in order for the changes to take effect.

It is possible that after a restart JCrypTool menus contain a mix of English and German entries. Please quit JCrypTool and switch to the @user.home/documents/.jcryptool directory in your file manager. Delete the complete .metadata directory in this directory. This resets all plug-in settings but keeps your crypto keys created in JCrypTool. You can delete the complete .jcryptool directory alternatively, but you will lose all your settings and data this way.


Remark:

You can start JCrypTool with the parameter -nl from the commandline (terminal) and so directly enforce a particular language.

Here is an example how the according commandline call looks like under Windows:
F:\Downloads\jcryptool\JCrypTool.exe -nl en
And here is a sample call under Linux:
$ ./JCrypTool -nl en &
