Editors

JCrypTool contains two different editors in the default installation: Text and Hex. While the text editor is capable of handling normal text files only, (almost) any file can be opened in the hex editor.

The result of a cryptographic operation is normally opened in an editor of the same type (text in the text editor and hex in the hex editor). Depending on the cryptographic operation it might be necessary to open the resulting file in the hex editor (in order to avoid display errors).

After opening a file in one of the editors it is possible to switch to the other editor using the Edit -> Open with menu.
