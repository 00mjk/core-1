Algorithm Perspective

The Algorithm Perspective offers a functions-oriented access to a default crypto library included in JCrypTool. The algorithms shown in this perspective update themselves automatically when updating the crypto library. So a new crypto library version may automatically lead to new and extended wizards.

For quite a few of the algorithms in the Algorithms Perspective, OID numbers are shown: These are globally unique identifiers in a hierarchical namespace defined by the ASN.1 standard that here denote crypto algorithms.


In the Algorithms Perspective, the main menu is shortened: The menus present in the Default Perspective (Algorithms, Analysis, Visuals, Games) are not available here.
