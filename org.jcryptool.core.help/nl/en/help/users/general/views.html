Views

JCrypTool offers a lot of different views. Most of them do display a certain cryptographic operation, like analysis or visualizations. These view open in half screen size or even full screen automatically. Other views, like the Error Log view for example, serve a special purpose only (in this case to display error and information messages depending on the selected log level in the Preferences).

You can close, resize and move any view. In case you closed a view accidentally you can reopen it using the menu Window -> Show View. A double click on the views tab maximizes it to full screen. Another double click on the tab minimizes the view again to its original size.

An overview of all views is available in the menu Window -> Show View -> Other. The appearing dialog contains all available views, sorted by category. Already visible views in this Perspective are shown inactive.

The most important views for cryptographic operations in JCrypTool are the File Explorer view and the Crypto Explorer view. For experts and those in a hurry there are the Console and the Actions view.

Fast Views

The status bar in the lower left of JCrypTool contains an icon, which lets you open a so called fast view. Fast views are normal views, but are automatically hidden when loosing focus. Each fast view shows an additional icon next to the default fast view icon.

Every view can be used as a fast view. A view can either be used as fast view or as normal view, not both at the same time. So in case you are showing the File Explorer as a fast view, the normal view is changed to a fast view. To change a fast view to a normal view again right click on the views fast view icon and deactivate the Fast View entry.
