Perspectives

Perspectives can be regarded as containers, in which other components, namely views and editors, are embedded and can be layouted in almost any way you desire.

JCrypTool offers two perspectives: Default and Algorithm. The Default perspective shows all possibilities in a common way with menus and views to the beginner. The Algorithm perspective is more suitable for advanced users and enables you to define every cryptographic operation in every possible detail.

Initially, JCrypTool has already defined the layout of both perspectives. In case of miss configuration or other problems you can use the menu Window -> Reset Perspective to go back to the initial configuration of that perspective.

After launching JCrypTool, always the last active perspective is shown. Use the buttons in the upper right of the main window, or the menu Window -> Open Perspective, to switch between the existing perspectives. The menu Window -> Customize Perspective opens a dialog in which you can select the elements (menu entries and toolbar icons) that should be displayed in the current perspective.

Views are independent in each perspective. Closing or moving a view in the Default perspective has absolutely no effect on the same view in the Algorithm perspective. Editors on the other side are shared between perspectives. Closing an editor in the Default perspective does close this editor in the Algorithm perspective as well.
