About JCrypTool

Release Candidate 5 - August 2011

JCrypTool is an extendable e-learning platform presenting cryptography, cryptanalysis and IT-Security in a modern and easy-to-use fashion. JCrypTool is built as platform-independent open source software and is part of the open source project CrypTool.

JCrypTool is licensed under the Eclipse Public License (EPL) version 1.0.

Visit http://jcryptool.sourceforge.net for more information about this cryptographic e-learning software. There you will find a presentation to make your first steps with JCrypTool as easy as possible (this presentation is intended for users and for developers).

The source code of all JCrypTool Core plug-ins is available at http://sourceforge.net/projects/jcryptool. The source code of most JCrypTool Plug-ins is available at http://sourceforge.net/projects/jctplugins.
