About JCrypTool

Release Candidate 5 - September 2011

JCrypTool is an extendable e-learning platform presenting cryptography, cryptanalysis and IT-Security in a modern and easy-to-use fashion. JCrypTool is built as platform-independent open source software and is part of the open source project CrypTool.

JCrypTool is licensed under the Eclipse Public License (EPL) version 1.0.

Visit https://github.com/jcryptool for more information about this cryptographic e-learning software. There you will find a presentation to make your first steps with JCrypTool as easy as possible (this presentation is intended for users and for developers).

The source code of all JCrypTool Core plug-ins is available at https://github.com/jcryptool/core. The source code of most JCrypTool Crypto Plug-ins is available at https://github.com/jcryptool/crypto.
