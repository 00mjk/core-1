Development

JCrypTool can be extended with own Plug-ins in almost any place. Use Features to group and organize plug-ins in order to make end users the installation of plug-ins as easy as possible.

Plug-ins and features can be installed in different ways in JCrypTool. The most easy way is to use a p2 Update Site. In case you do not have any web space available you can offer a downloadable (zip-)archive. Users can extract this archive into the dropins folder of their JCrypTool installation directory.
