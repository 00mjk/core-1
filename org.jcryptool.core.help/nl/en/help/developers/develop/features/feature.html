Features

JCrypTool recommends feature usage. However it is possible to only develop plug-ins and to install them separately in JCrypTool. JCrypTool itself is segmented in different features. All feature projects end on .feature and in case of crypto plug-ins do contain the name of the category as well. For example org.jcryptool.crypto.modern.feature.

To use an Update Site for plug-in maintenance you have to use features, never plug-ins. Even in case only one plug-in changed in the whole feature set.
