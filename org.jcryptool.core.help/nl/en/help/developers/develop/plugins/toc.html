Plug-ins

Plug-ins are the common way to extend JCrypTool. Based on the type of plug-in menus and toolbars, help contents, preferences or other parts of the GUI will be extended with new functionality.

No matter what you plan to extend, please have a look at the hints on internationalization and persistence.

First Steps

JCrypTool contains and offers the same extension points as Eclipse (e.g. for views). However, it is not enough to extend these extension points to add a new crypto plug-in to JCrypTool. To add e.g. a new visualization to the Crypto Explorer (and at the same time to the menu), the special JCrypTool Extension Point org.jcryptool.core.operations.visuals must be extended as well (or the corresponding extension points for analysis and other cryptographic operations).

After that, the runtime configuration (the entry in the run menu) must be extended. The new plug-in has to be added on the Plug-ins tab. For exporting, the corresponding Feature must be updated by adding the plug-in there as well.
