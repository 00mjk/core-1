General Information

JCrypTool is based on Eclipse (Luna Release - version 4.4) and Java 1.7 and can be extended by anyone with custom plug-ins and features. These extensions can be inserted at every place within the whole e-learning platform.

To ensure a simple handling and to present a consistent appearance the JCrypTool core developers published guidelines for all plug-in developers. These guidelines of JCrypTool core consist of rules on how to extend the Help Content, Preference Pages, Extension Points and some more. Plug-ins which do violate these guidelines won't be included in our official release and won't be listed on our official JCrypTool Update Site.

Besides some Conventions are all development guidelines included in the Development area. In addition to these guidelines all developers are supposed to follow the Code Conventions for Java and the Eclipse Development Conventions and Guidelines.
