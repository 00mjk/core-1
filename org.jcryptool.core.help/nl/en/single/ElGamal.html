ElGamal

ElGamal is a public-key cryptosystem which is based on the Diffie-Hellman key exchange and on the difficulty of computing discrete logarithms. It can be used for encryption and in a variant as signature scheme.

This dialog allows to encrypt and decrypt with ElGamal using a key from the JCT keystore.
