SHA-256

Description

SHA-2 (Secure Hash Algorithm 2) is a set of 6 cryptographic hash functions designed by the United States National Security Agency (NSA) and first published in 2001. SHA-2 is still in use, besides the NIST hash function competition selected a new hash function, SHA-3, in 2012. Some SHA-3 candidates of this competeitions are also available in JCT: see SHA-3 Candidates.
