Entwicklung

JCrypTool kann nahezu nach Belieben mit eigenen Plug-ins erweitert werden. Um den Anwendern die Installation von Plug-ins zu erleichtern werden diese in der Regel zu Features zusammengefasst.

Plug-ins und Features können dabei auf verschiedene Arten in JCrypTool installiert werden. Die einfachste Variante ist die Installation über den integrierten Update Manager. Alternativ können Sie auch ein (zip-)Archiv zum Download anbieten. Anwender können dieses dann einfach im dropins Ordner im JCrypTool Installationsverzeichnis entpacken.
