Extension Points

JCrypTool verfügt über eine stetig wachsende Anzahl an Extension Points. Extension Points stellen dabei die Punkte dar, an denen JCrypTool am besten erweitert werden kann. Die meisten (und wohl wichtigsten) Extension Points finden sich im org.jcryptool.core.operations Plug-in: von den Algorithmen bis hin zu den Providern.

Extension Points werden grundsätzlich in einer *.exsd Datei definiert. Diese befindet sich in der Regel in einem schemas Verzeichnis im definierenden Plug-in. Einen JCrypTool Extension Point verwenden Sie dabei genauso wie einen normalen Eclipse Extension Point: Öffnen Sie die plugin.xml Ihres Plug-ins und klicken Sie auf den Add Button auf der Extensions Seite des Editors. Alle JCrypTool Extension Points beginnen mit org.jcryptool als ID, z.B. org.jcryptool.core.algorithms.

Dokumentation zum Extension Point

Fügen Sie den Extension Point wie oben beschrieben zu Ihrem Plug-in hinzu. Nach dem Markieren des neuen Eintrags erscheint ein Link Show extension point description auf der rechten Seite. Ein Klick darauf öffnet ein weiteres Fenster mit Informationen zum gewählten Extension Point. Über weitere Links im Editor können Sie zum Extension Point Schema springen oder das deklarierende Plug-in suchen.

Wichtige Extension Points

Die für kryptografische Erweiterungen (Krypto Plug-ins) notwendigen Extension Points gehören zu den meist verwendeten in JCrypTool:

  • org.jcryptool.core.operations.algorithms
  • org.jcryptool.core.operations.analysis
  • org.jcryptool.core.operations.games
  • org.jcryptool.core.operations.visuals

Einer dieser Extension Points (abhängig vom Typ Ihres Plug-ins) ist notwendig um JCrypTool um ein weiteres Krypto Plug-in zu erweitern. Der Extension Point sorgt dafür, dass die kryptografische Erweiterung (z.B. ein Algorithmus) im entsprechenden Menü auftaucht und auch die Krypto-Explorer Sicht um einen Eintrag erweitert wird. D.h. Sie müssen nichts weiter tun als die mit einem * markierten Pflichtfelder auszufüllen und schon zeigt JCrypTool Ihre Erweiterung in der GUI an.
