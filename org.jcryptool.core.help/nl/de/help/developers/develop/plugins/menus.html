Menüs und Toolbars

Bei Verwendung eines der JCrypTool Extension Points müssen Sie sich über die Erweiterung des Menüs (z.B. Algorithmen) und über die Erweiterung der Krypto-Explorer Sicht keine Gedanken machen. JCrypTool verwendet für alle Krypto Plug-ins eine Art Pull Mechanismus, d.h. JCrypTool selbst sucht nach Krypto Plug-ins die in die jeweilige Kategorie passen. Dies weicht vom üblichen Eclipse Plug-in Mechanismus ab, da normalerweise die Plug-ins selbst bestimmen, wo Menüs, Toolbars oder andere Teile der GUI erweitert werden.

Prinzipiell ist JCrypTool überall erweiterbar. Der hier beschriebene Mechanismus greift allerdings nur für Krypto Plug-ins (gleich welcher Kategorie). Sofern Sie ein anderes Plug-in zu JCrypTool hinzufügen möchten (z.B. einen Editor, eine Sicht, ...) muss der in Eclipse übliche Mechanismus verwendet werden.

Die Datei-Explorer Sicht erweitern

Die Datei-Explorer Sicht kann umfangreich an vielen Punkten erweitert werden. Neben dem Kontextmenü lassen sich so auch das Menü der Sicht sowie deren Toolbar erweitern.

Kontextmenü

Die Kontextmenü (Popup Menü) ID lautet org.jcryptool.fileexplorer.popup. Dieses Menü kann an fünf Positionen erweitert werden:

  • crypto
  • edit
  • delete
  • rename
  • additions

Für Kryptografie Plug-ins ist der Bereich crypto vorgesehen. Um Beispielsweise hier einen zusätzlichen Eintrag hinzuzufügen muss die folgende Location URI (für Eclipse Commands) verwendet werden: popup:org.jcryptool.fileexplorer.popup?before=crypto. Neben before ist auch noch after möglich. Contributions ohne exakte Positionsangabe, d.h. nur mit popup:org.jcryptool.fileexplorer.popup werden ganz am Ende des Menüs (im Bereich additions) angefügt.

Kryptografie Plug-ins, die ohnehin den algorithms Extension Point im org.jcryptool.core.operations Plug-in erweitern, werden automatisch mit in das Menü Verschlüsselung/Entschlüsselung aufgenommen. Ein solches Plug-in darf keinen weiteren Eintrag im Kontextmenü anlegen!

Menü

Um das Menü der Datei-Explorer Sicht zu erweitern muss die Location URI menu:org.jcryptool.fileexplorer.views.FileExplorerView verwendet werden.

Toolbar

Um die Toolbar der Datei-Explorer Sicht zu erweitern muss die Location URI toolbar:org.jcryptool.fileexplorer.views.FileExplorerView angegeben werden.
