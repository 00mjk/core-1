Logging

Um das Logging für Entwickler so einfach wie möglich zu halten und für Anwender nur ein einziges Log anzuzeigen, gibt es das org.jcryptool.core.logging Plug-in. Alle Logmeldungen werden dabei in das Workspace-Log geschrieben und können über die Fehlermeldungen Sicht angezeigt werden. Welches Loglevel verwendet wird kann der Anwender in den Benutzereinstellungen festlegen. Nur Meldungen ab diesem Level werden protokolliert, diese Einstellung wirkt sich somit nicht auf den Filter der Sicht aus. Standardeinstellung ist das Level Fehler (Error).

Entwickler können damit einfach die LogUtil Methoden aufrufen und müssen keine Prüfung auf das gerade aktive Loglevel vornehmen. Ob eine Meldung protokolliert wird oder nicht entscheidet sich damit alleine im Logging Plug-in.

Um für ein Plug-in das Logging zu aktivieren muss über das MANIFEST.MF das Package org.jcryptool.core.logging.utils zu den Required Bundles hinzugefügt werden. Anschließend kann auf die verschiedenen statischen Logmethoden über die Klasse LogUtil zugegriffen werden:

LogUtil.logError(bundleId, message, exception);

Da es sich hier um eine technische Fehlermeldung handelt sollte diese ausschließlich in Englisch protokolliert werden (und nicht internationalisiert sein). Diese Fehlermeldung dient nur dem Logging und soll die spätere Fehlersuche erleichtern.

Ausnahme hiervon ist die mit vier Parametern aufzurufende logError Methode:

LogUtil.logError(bundleId, message, exception, true);

Hier steuert der letzte boolean Parameter, ob dem Anwender mit der Protokollierung gleichzeitig ein Fehlerdialog angezeigt werden soll. Für diese Methode müssen alle Parameter angegeben werden. Da die Fehlermeldung nun sowohl im Log als auch im Dialog für den Anwender angezeigt wird, muss diese internationalisiert werden! Des Weiteren muss die Fehlermeldung auch für normale Anwender verständlich sein, und einen Hinweis zur Behebung des Fehlers beinhalten (sofern möglich). Die Verwendung dieser Methode wird empfohlen, um überall in JCrypTool einheitliche Fehlerdialoge anzuzeigen.
