Benutzervorgaben

Auch die Benutzervorgaben von JCrypTool können von allen Plug-ins erweitert werden. Jedes Plug-in steuert dabei seine eigene Seite bei. Es ist somit nicht möglich, eine von einem anderen Plug-in erzeugte Preference Page durch ein anders Plug-in zu erweitern.

<extension point="org.eclipse.ui.preferencePages">

  category="org.jcryptool.preferences.general"
  class="org.jcryptool.actions.ui.preferences.GeneralPage"
  id="org.jcryptool.actions.ui.general"
  name="Actions">
 </page>
</extension>

Krypto Plug-ins

Die Gruppierung der Benutzervorgaben spiegelt die in JCrypTool überall verwendete Kategorisierung wieder. D.h. es gibt Bereiche für Algorithmen, Analysen, Spiele und Visualisierungen. Es ist die Aufgabe jedes Plug-in Entwicklers, die Seite(n) für die Benutzereinstellungen in die korrekte Kategorie einzufügen.

Kryptographie

Dies ist der Bereich für allgemeine kryptografische Benutzervorgaben. Die ID für diese Kategorie lautet org.jcryptool.preferences.crypto.

Algorithmen

Dies ist der Bereich für Algorithmen Plug-ins, von klassischen bis hin zu modernen Verfahren. Die ID für diese Kategorie lautet org.jcryptool.preferences.algorithms.

Analysen

Dies ist der Bereich für kryptografische Analysen. Die ID für diese Kategorie lautet org.jcryptool.preferences.anlysis.

Spiele

Dies ist der Bereich für kryptografische Spiele. Die ID für diese Kategorie lautet org.jcryptool.preferences.games.

Visualisierungen

Dies ist der Bereich für kryptografische Visualisierungen. Die ID für diese Kategorie lautet org.jcryptool.preferences.visuals.

Sonstige Plug-ins

Auch die weiteren allgemeinen Seiten in den Benutzereinstellungen können ergänzt werden.

Allgemein

In diesen Bereich gehören die allgemeinen Einstellungen für JCrypTool. Die ID für diese Kategorie lautet org.jcryptool.preferences.general.

Editoren

Editoren gleich welchen Typs gehören in diese Kategorie. Die ID für diese Kategorie lautet org.jcryptool.preferences.editors.
