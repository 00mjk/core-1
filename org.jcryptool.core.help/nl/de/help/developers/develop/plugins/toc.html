Plug-ins

Plug-ins sind der ideale Weg, um JCrypTool zu erweitern. Je nach Art des Plug-ins werden dabei Menüs und Toolbars, die Hilfe, Benutzereinstellungen oder andere Stellen in der Oberfläche erweitert.

Beachten Sie dabei auf alle Fälle die Hinweise zur Internationalisierung und zur Persistenz.

Erste Schritte

JCrypTool besitzt grundsätzlich dieselben Erweiterungsmöglichkeiten wie Eclipse auch (z.B. für Sichten). Allerdings genügt das Erweitern dieser Extension Points nicht, um ein neues Crypto Plug-in in JCrypTool auch sichtbar zu machen. Damit z.B. eine neue Visualisierung im Krypto-Explorer (und gleichzeitig auch dem Menü) angezeigt wird, muss zusätzlich der JCrypTool eigene Extension Point org.jcryptool.core.operations.visuals erweitert werden (für Analysen und andere kryptografische Operationen gelten die entsprechenden Erweiterungspunkte).

Anschließend muss die Runtime Konfiguration (der Eintrag im Run Menü) auf dem Tab Plug-ins um das neue Plug-in erweitert werden. Damit auch ein Export das neue Plug-in enthält, muss das entsprechende Feature aktualisiert und das neue Plug-in dort hinzugefügt werden.
