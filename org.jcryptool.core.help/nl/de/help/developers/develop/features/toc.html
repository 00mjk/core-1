Features

Features sind Gruppierungen von zusammengehörigen Plug-ins. D.h. ein Feature enthält keinerlei Code, sondern ausschließlich eine Art Liste in XML-Form, in der die im Feature enthaltenen Plug-ins (und deren Voraussetzungen) aufgelistet werden. Features sind damit hauptsächlich ein Hilfsmittel, um dem Endanwender die Installation und Verwaltung zu erleichtern.
