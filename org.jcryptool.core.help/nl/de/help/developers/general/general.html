Allgemeine Informationen

JCrypTool basiert auf Eclipse (Indigo Release - Version 3.7) und kann daher von jedem Entwickler um eigene Plug-ins und Features erweitert werden. Prinzipiell können diese Erweiterungen an nahezu jeder Stelle in der gesamten eLearning Plattform eingefügt werden.

Um die JCrypTool Bedienung möglichst einfach und einheitlich zu gestalten sowie ein rundes Gesamtbild zu präsentieren, werden vom JCrypTool Core Team gewisse Vorgaben gemacht, an die sich Plug-in Entwickler halten sollten. Neben den vom JCrypTool Core definierten Extension Points sind dies u.a. auch Vorgaben, an welchen Punkten die Hilfe Inhalte zu erweitern sind, wo Preference Pages erscheinen sollen und einige wenige Punkte mehr. Plug-ins, die gegen diese Vorgaben verstoßen, werden nicht in die offiziellen JCrypTool Downloads aufgenommen und werden auch nicht auf der offiziellen JCrypTool Update Site gelistet.

Neben einigen JCrypTool Konventionen finden sich sämtliche Entwicklungsrichtlinien im Bereich Entwicklung. Neben diesen JCrypTool eigenen Vorgaben gelten ansonsten die Code Conventions for Java und die Eclipse Development Conventions and Guidelines.
