Lizenzen

Es gibt zahlreiche Open Source Lizenzen, unter denen Sie Ihr JCrypTool Plug-in veröffentlichen können. Alle unsere Plug-ins im JCrypTool Core oder Crypto Repository stehen jedoch unter der Eclipse Public License (EPL) Version 1.0. Wie gesagt, es steht Ihnen frei eine andere Lizenz zu wählen. Soll Ihr Plug-in jedoch in einem der offiziellen JCrypTool Projekte auf GitHub gelistet werden, muss dieses zwingend unter der EPL stehen. Dazu muss auch der folgende Disclaimer am Beginn jeder Java Klasse stehen!

Voraussetzungen

Damit der EPL genüge getan wird, muss jede Java Klasse einen Hinweis auf diese Lizenz enthalten. Wir verwenden dazu den folgenden Disclaimer (den Sie für die Übernahme Ihrer Sourcen auf GitHub übernehmen müssen):
