Eclipse Plug-ins

JCrypTool enthält bereits in der Standardinstallation eine Reihe von Eclipse Plug-ins (das Vorhandensein dieser Plug-ins wird von JCrypTool garantiert). Auf diese können Entwickler wie in jeder anderen Eclipse Umgebung zugreifen. In der Regel wird immer die aktuellste Version im JCrypTool enthalten sein. Lediglich hier nicht enthaltene Plug-ins oder speziell benötigte Versionen (Versionsabhängigkeit sollte grundsätzlich vermieden werden) müssen in Ihrem eigenen Plug-in mitgeliefert werden.

D.h. sobald Sie für Ihr Plug-in ein Feature zusammenstellen sollten Sie überprüfen, welche Eclipse Plug-ins bereits im plugins Verzeichnis von JCrypTool enthalten sind. Diese Plug-ins müssen Sie nicht zusätzlich mit Ihrem Feature ausliefern (Sie können das zwar ohne weitere negativen Konsequenzen tun, erhöhen dadurch aber die Feature-Größe unnötig). Lediglich hier nicht vorhandene Plug-in Abhängigkeiten müssen Sie mit Ihrem Feature ausliefern damit Ihr Plug-in später auch funktionsfähig ist.

Eine komplette Übersicht über alle in einer JCrypTool Version enthaltenen Eclipse Plug-ins erhalten Sie über das Feature Projekt org.jcryptool.core.dependencies.feature. Hier finden Sie unter Plug-ins und Included Features eine vollständige Liste. Im Feature Projekt org.jcryptool.core.dependencies.nl_de.feature sind zusätzlich sämtliche deutschen Sprachpakete enthalten.
