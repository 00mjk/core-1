JCrypTool-Hilfe

Willkommen in der JCrypTool Online-Hilfe!

Das Anwenderhandbuch in dieser Online-Hilfe gibt eine Einführung in JCrypTool (JCT). Es erklärt die Funktionsweise und Bedienung der verschiedenen Editoren, Visualisierungen und Plug-ins zur Verschlüsselung, Signierung, usw.
Dies ist der ideale Einstiegspunkt, wenn Sie noch nie mit JCrypTool gearbeitet haben.

Um das Anwenderhandbuch zu öffnen, klicken Sie im Inhaltsverzeichnis auf der linken Seite auf den Eintrag Anwenderhandbuch. Jetzt sehen Sie die verschiedenen Kategorien, in denen die Hilfen im JCT kategorisiert sind, bspw. Allgemeine Informationen oder Kryptologie. Diese können Sie mit einem Klick auf den jeweiligen Eintrag öffnen. So können Sie sich bis zur gesuchten Hilfeseite durchnavigieren.

Alternativ können Sie das Suchfeld im Linken oberen Bereich der Seite nutzen, um Hilfe zu einem bestimmten Thema zu bekommen. Suchen Sie dafür am besten nach dem Namen eines Plugins oder Verfahrens.

Entwickler finden für sie relevante und aktuelle Informationen in unserem GitHub Wiki.

Weitere Informationen zu JCT finden Sie auf unserer Homepage. Der Source Code zu JCrypTool ist auf GitHub frei erhältlich.
