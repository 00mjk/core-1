JCrypTool Hilfe

Willkommen in der JCrypTool Online-Hilfe! Die Hilfe zur JCrypTool E-Learning-Plattform besteht aus zwei Bereichen:

  • Das Anwenderhandbuch gibt eine Einführung in die Funktionen JCrypTools, von den verschiedenen Editoren über die Visualisierungen bis hin zu den Plug-ins zur Verschlüsselung, Signierung, usw.
    Dies ist der ideale Einstiegspunkt, wenn Sie noch nie mit JCrypTool gearbeitet haben.
  • Das Entwicklerhandbuch enthält Informationen für Entwickler und fortgeschrittene Anwender, die JCrypTool um eigene Plug-ins erweitern möchten.

Wie das gesamte JCrypTool ist auch die Online-Hilfe modular erweiterbar. Jedes Kryptografie-Plug-in ist selbst für seine Hilfe im Anwenderhandbuch verantwortlich. Wenn Sie zu einem Plug-in keine Hilfe finden, ist es wahrscheinlich, dass das Plug-in keine zur Verfügung stellte.

Weitere Informationen zu JCrypTool finden Sie auf unserer Homepage. Der Source Code zu JCrypTool Core und zu JCrypTool Crypto ist frei erhältlich.
