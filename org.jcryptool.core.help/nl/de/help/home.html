JCrypTool Hilfe

Willkommen in der JCrypTool Online-Hilfe! Die umfangreiche Hilfe zur JCrypTool E-Learning Plattform besteht aus vier Bereichen:

  • Das Anwenderhandbuch gibt eine Einführung in die Möglichkeiten JCrypTools, von den verschiedenen Editoren bis hin zu den einzelnen Plug-ins zur Verschlüsselung, Signierung, usw. Dies ist der ideale Einstiegspunkt wenn Sie noch nie mit JCrypTool gearbeitet haben...
  • Das Entwicklerhandbuch enthält Informationen für Entwickler und fortgeschrittene Anwender, die JCrypTool um eigene Plug-ins erweitern möchten...
  • Kryptologie enthält die theoretischen Informationen rund um Kryptografie, Grundlagen zu Algorithmen, der Verschlüsselung, digitalen Signaturen und vielem mehr...
  • Das Glossar erklärt in Kurzform verschiedene kryptografische Begriffe...

Wie das gesamte JCrypTool ist auch die Online-Hilfe umfangreich erweiterbar. Und wie bei den Plug-ins ist auch in der Hilfe jedes Kryptografie Plug-in selbst für diesen Teil verantwortlich. Wenn Sie zu einem Plug-in daher keine Hilfe finden können ist es wahrscheinlich, dass das Plug-in keine zur Verfügung stellt. Ebenso ist es möglich, dass die Hilfe nicht in den beiden Sprachen Deutsch und Englisch verfügbar ist.

Weitere Informationen und Unterstützung zum JCrypTool finden Sie auf unserer Homepage. Auch der JCrypTool Core Source Code sowie der JCrypTool Crypto Source Code sind frei erhältlich.
