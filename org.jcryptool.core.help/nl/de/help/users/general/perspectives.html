Perspektiven

Perspektiven sind eine Art Container, in denen die untergeordneten Komponenten, das sind Sichten und Editoren, von Ihnen nahezu beliebig angepasst werden können.

JCrypTool stellt Ihnen zunächst einmal zwei Perspektiven zur Verfügung: Standard und Algorithmen. Dem Einsteiger bietet die Standard-Perspektive alle Möglichkeiten von klassischen bis hin zu modernen Verfahren an. Fortgeschrittene Anwender können in der Algorithmen-Perspektive die FlexiProvider Verfahren bis ins kleinste Detail einstellen.

Initial gibt JCrypTool das Layout dieser beiden Perspektiven vor. Bei Bedarf können Sie dieses Layout aber nahezu nach Belieben anpassen und nur von Ihnen gewünschte Sichten an den von Ihnen gewünschten Positionen anzeigen. Sollten Sie einmal eine Perspektive beim Konfigurieren unbrauchbar gemacht haben, können Sie diese über das Menü Fenster -> Perspektive zurücksetzen wieder in ihren Ursprungszustand zurücksetzen lassen.

Beim Start von JCrypTool wird Ihnen grundsätzlich die zuletzt geöffnete Perspektive angezeigt. Der Wechsel zwischen den beiden Perspektiven erfolgt über die beiden Icons Standard und Algorithmen rechts oben oder über das Menü Fenster -> Perspektive öffnen. Im Menü Fenster -> Perspektive anpassen können Sie in einem Dialog auswählen, welche Elemente (d.h. Menüeinträge und Icons der Toolbar) in der aktiven Perspektive angezeigt werden sollen. Diese Änderungen haben nur Auswirkungen auf die gerade aktive Perspektive.

Sichten sind in jeder Perspektive unabhängig vorhanden. Das Schließen oder Verschieben einer Sicht in der Standard-Perspektive hat so keinen Einfluss auf dieselbe Sicht in der Algorithmen-Perspektive. Editoren dagegen sind in jeder Perspektive identisch. Das Schließen eines Editors in der Standard-Perspektive schließt somit auch denselben Editor in der Algorithmen-Perspektive.
