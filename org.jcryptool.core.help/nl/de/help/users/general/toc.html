Allgemeine Informationen

JCrypTool basiert auf der Eclipse Rich Client Platform (RCP), wodurch Eclipse Konzepte wie Perspektiven, Sichten und Editoren eine sehr wichtige Rolle spielen. Mit diesen Begriffen und den dahinterstehenden Konzepten können Sie sich auf den folgenden Seiten, soweit für JCrypTool notwendig, vertraut machen.

Um die Verwendung von JCrypTool so einfach wie möglich zu machen finden Sie in beiden Perspektiven links unten eine Sicht mit einer kontextsensitiven Kurzhilfe. Hier werden Ihnen übersichtliche Information zum gerade aktiven Element angezeigt. Über die in diesem Text enthaltenen Links können Sie weitergehende Informationen abrufen. Bitte beachten Sie, dass die Kontexthilfe von jedem Plug-in selbst beigesteuert werden muss. Sofern diese Information von einem Plug-in (noch) nicht geliefert wird zeigt die Sicht daher nur allgemeine Informationen an.

In vielen Assistenten und Dialogen erhalten Sie die vorgestellte Kontexthilfe über das Hilfe-Icon (dargestellt durch ein ?) links unten. Auch hier muss diese Information durch das jeweilige Plug-in zur Verfügung gestellt werden, die Kontexthilfe muss damit nicht in jedem Dialog vorhanden/ aktiv sein.
