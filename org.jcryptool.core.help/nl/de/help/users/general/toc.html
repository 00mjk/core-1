Allgemeine Informationen

JCrypTool basiert auf Eclipse, wodurch Eclipse-Konzepte wie Perspektiven, Sichten und Editoren eine sehr wichtige Rolle für die Bedienung der GUI spielen. Mit diesen Begriffen und den dahinterstehenden Konzepten können Sie sich auf den folgenden Seiten, soweit für JCrypTool notwendig, vertraut machen.

Zugang zur Onlinehilfe

Um die Verwendung von JCrypTool so einfach wie möglich zu machen, zeigt das Hauptfenster in beiden Perspektiven links unten eine Sicht mit einer kontextsensitiven Kurzhilfe. Hier werden Ihnen übersichtliche Information zum gerade aktiven Element angezeigt.

Diese Kontexthilfe erhalten Sie durch Drücken der Taste F1.

Die ausführliche Onlinehilfe wird in einem extra Browser-Fenster anzeigt. Diese rufen Sie auf über das Hilfe-Icon (dargestellt durch ein ?) oben im Haupt-Toolbar.

Sowohl die Kontexthilfe als auch die ausführliche Onlinehilfe werden von jedem Plug-in selbst beigesteuert. Sofern der Plug-in-Autor diese Information (noch) nicht lieferte, werden nur allgemeine Informationen angezeigt.
