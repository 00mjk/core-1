Web Browser

Die Web Browser Sicht bietet Ihnen die Möglichkeit, Webseiten in JCrypTool mit Ihrem Standard Web Browser anzuschauen. JCrypTool bringt also keinen eigenen Web Browser mit, sondern integriert den in Ihrem System aktiven Standard Web Browser und stellt diesen als Sicht in JCrypTool dar. Damit haben Sie einfach und schnell die Möglichkeit, kryptografische Webseiten (andere natürlich auch) aufzurufen und das dort Gezeigte einfach in JCrypTool auszuprobieren.
