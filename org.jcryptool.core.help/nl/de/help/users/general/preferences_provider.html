Krypto-Anbieter

Auf dieser Seite erhalten Sie zum einen eine Übersicht über alle derzeit in JCrypTool installierten Krypto-Anbieter und deren Versionen. Zum anderen können Sie über die Hoch und Runter Buttons auf der rechten Seite die Reihenfolge der Krypto-Anbieter festlegen. Der Anbieter an erster Stelle wird dabei automatisch der Standardanbieter.

Wählen Sie nun in einem der Krypto Plug-ins einen bestimmten Algorithmus aus versucht JCrypTool, eine Implementierung für diesen zu finden. Dazu werden die installierten Krypto-Anbieter in der von Ihnen festgelegten Reihenfolge durchsucht, so lange, bis eine passende Implementierung gefunden wird. Bietet nun mehr als ein Krypto-Anbieter eine Implementierung für einen Algorithmus an können Sie so festlegen, welcher Anbieter verwendet werden soll. Bietet keiner eine passende Implementierung an erhalten Sie eine Fehlermeldung.

Standard-Krypto-Anbieter ist in JCrypTool der FlexiProvider. Zusätzlich ist BouncyCastle in jeder Installation enthalten.
