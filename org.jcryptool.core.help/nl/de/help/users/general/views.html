Sichten (Views)

JCrypTool verfügt über eine ganze Reihe unterschiedlicher Sichten (Views). Viele sind spezialisiert auf bestimmte kryptografische Operationen, z.B. die Analysen und Visualisierungen. Diese Sichten nehmen beim Aufruf automatisch die Hälfte oder den ganzen Bildschirm ein. Andere Sichten, wie beispielsweise die Fehlerprotokoll Sicht, dienen ganz speziellen Zwecken (in diesem Fall der Fehleranzeige basierend auf dem in den Benutzervorgaben gewählten Log-Level).

Sichten können von Ihnen geschlossen, vergrößert oder verkleinert und auch verschoben werden. Sollten Sie eine Sicht versehentlich schließen können Sie diese über das Menü Fenster -> Sicht anzeigen wieder einblenden. Per Doppelklick auf den Tab der Sicht wird diese zu einer Vollbilddarstellung maximiert. Ein weiterer Doppelklick auf den Tab setzt die Sicht auf ihre ursprüngliche Größe zurück.

Eine Übersicht aller Sichten finden Sie im Menü Fenster -> Sicht anzeigen -> Andere. Der nun erscheinende Dialog enthält alle vorhandenen Sichten, sortiert nach Kategorie. Bereits angezeigte Sichten in dieser Perspektive werden inaktiv dargestellt und können nicht erneut eingeblendet werden.

Die für kryptografische Operationen aller Art wohl wichtigsten Sichten in JCrypTool sind die Datei-Explorer Sicht und die Krypto-Explorer Sicht. Für Experten und ganz Eilige existiert die Konsole und auch die Actions Sicht.

Schnellsichten (Fast Views)

In der Statuszeile links unten in JCrypTool finden Sie ein Icon, mit dem Sie eine so genannte Schnellsicht öffnen können. Schnellsichten sind normale Sichten, werden aber automatisch ausgeblendet sobald sie den Fokus verlieren. Jede aktivierte Schnellsicht blendet dabei ein zusätzliches Icon neben dem Schnellsicht Icon ein.

Jede Sicht kann als Schnellsicht angezeigt werden. Eine Sicht kann jedoch entweder normal oder als Schnellsicht angezeigt werden, nicht beides gleichzeitig. D.h. wenn Sie z.B. den Datei-Explorer als Schnellsicht auswählen wird die bisherige Sicht ausgeblendet und nur noch als Schnellsicht angezeigt. Um eine Schnellsicht wieder zu einer normalen Sicht zu machen klicken Sie mit der rechten Maustaste auf das Schnellsicht Icon der gewünschten Sicht und deaktivieren den Eintrag Schnellsicht.
