Wechseln der Sprache

Die beim Start von JCrypTool gewählte Sprache entspricht den im Betriebssystem festgelegten Regionaleinstellungen. Ein deutsches Betriebssystem führt somit meistens zu einem deutschsprachigen JCrypTool.

Zum Wechseln der JCrypTool-Sprache öffnen Sie die Benutzervorgaben und wechseln auf die Seite Allgemein. In der dort angezeigten Auswahlbox können Sie die Sprache festlegen. Damit die Einstellungen aktiv werden, muss JCrypTool neu gestartet werden.

Nach dem Neustart kann es vorkommen, dass die Menüs einen Mix aus deutschen und englischen Einträgen anzeigen. In diesem Fall beenden Sie JCrypTool und löschen im Dateimanager das Verzeichnis .metadata im JCrypTool-Workspace-Verzeichnis @user.home/documents/.jcryptool. Damit setzen Sie alle Plug-ins zurück, Ihre Schlüssel und andere JCrypTool-Dateien bleiben erhalten. Sie können auch das gesamte .jcryptool Verzeichnis löschen, verlieren dann aber alle Einstellungen und Dateien.


Anmerkung:

Sie können JCrypTool auf der Kommandozeile (Terminal) mit dem Parameter -nl starten und so direkt eine bestimmte Sprache erzwingen.

Hier ist ein Beispiel, wie der Kommandozeilen-Aufruf unter Windows aussieht:
F:\Downloads\jcryptool\JCrypTool.exe -nl de
Und hier ein Beispielaufruf unter Linux:
$ ./JCrypTool -nl de &
