Wechseln der Sprache

Die beim ersten Start von JCrypTool gewählte Sprache entspricht den im Betriebssystem festgelegten Regionaleinstellungen. Ein deutsches Windows führt somit meistens zu einem deutschsprachigen JCrypTool. Manche Plug-ins können dennoch mit englischen Texten angezeigt werden, Ursache hierfür sind fehlende deutsche Übersetzungen.

Zum Wechseln der JCrypTool Sprache öffnen Sie die Benutzervorgaben und wechseln auf die Seite Allgemein. In der hier angezeigten Auswahlbox können Sie die Sprache festlegen. Damit die Einstellungen aktiv werden muss JCrypTool neu gestartet werden.

Nach dem Neustart kann es vorkommen, dass die Menüs einen Mix aus deutschen und englischen Einträgen anzeigen. Bei manchen Plug-ins ist es wie gesagt möglich, dass nur eine englische Version zur Verfügung steht. Die meisten Plug-ins liegen jedoch zweisprachig vor. In diesem Fall beenden Sie JCrypTool und löschen im Dateimanager das Verzeichnis .metadata im JCrypTool-Workspace-Verzeichnis @user.home/documents/.jcryptool. Damit setzen Sie alle Plug-ins zurück, Ihre Schlüssel und andere JCrypTool-Dateien bleiben allerdings erhalten. Sie können auch das gesamte .jcryptool Verzeichnis löschen, verlieren dann aber alle Einstellungen und Dateien.
