Benutzervorgaben

Die Benutzervorgaben werden über Fenster -> Benutzervorgaben aufgerufen. Aufgeteilt sind die Einstellungen in einzelne Kategorien, wobei sich viele der in JCrypTool üblichen Kategorien (wie Algorithmen, Visualisierungen) auch in den Benutzervorgaben wiederfinden. Beachten Sie, dass nicht jedes Plug-in über Benutzervorgaben verfügt. Manche Einstellungen können Sie stattdessen direkt im Assistent oder in der Sicht des jeweiligen Plug-ins festlegen.

Am schnellsten finden Sie die Einstellungen für ein Plug-in über das Suchfeld. Tragen Sie hier einfach einen passenden Begriff ein und wählen die korrekte Seite aus den Ergebnissen aus. In welchem Bereich ein Krypo Plug-in seine Benutzervorgaben ablegt hängt von der üblichen Kategorie des Plug-ins ab. D.h. ein Krypto Spiel werden Sie wie im Hauptdialog auch in der Kategorie Spiele finden. Weitere Kategorien sind die bereits bekannten Algorithmen, Analysen und Visualisierungen.

Neben den einzelnen Seiten für Plug-ins finden Sie die allgemeinen JCrypTool Einstellungen ebenfalls in den Benutzervorgaben. So können Sie hier beispielsweise die in JCrypTool aktive Sprache festlegen. Diese wird initial automatisch anhand Ihrer Betriebssystemeinstellungen festgelegt, kann von Ihnen aber auch explizit in den Benutzervorgaben ausgewählt werden. Nach dem Sprachwechsel muss JCrypTool neu gestartet werden.

Logging

Auf der Seite Logging können Sie die aktive Protokollierungsstufe festlegen. Standard ist hier Fehler, d.h. nur kritische Fehler werden in die JCrypTool Logdatei eingetragen. Warnungen protokolliert zusätzlich unkritische Fehler, Info protokolliert alle Informationen. Der Info-Level ist in der Regel nur für Entwickler interessant, die ihr Plug-in testen möchten und dazu viele Informationen benötigen. Sie sollten das Log-Level nur bei Bedarf ändern (beispielsweise um mehr Informationen zu einem Fehler zu bekommen) und anschließend wieder auf den Standardwert Fehler zurücksetzen. Das Protokollieren im Info-Level produziert nicht nur sehr viele Log-Einträge (und damit eine große Log-Datei), sondern verlangsamt JCrypTool durch die zusätzlichen Informationen auch.

Das hier generierte Protokoll finden Sie am einfachsten über die Fehlerprotokoll Sicht. Rufen Sie dazu Fenster -> Sicht anzeigen aus. Im Dialog finden Sie das Fehlerprotokoll in der Kategorie Allgemein.

Editoren

In der Kategorie Editoren finden Sie Einstellungen für die in JCrypTool vorhandenen Editoren.

Kryptografie

Hier befinden sich allgemeine Einstellungen für Krypto Plug-ins. Diese Einstellungen sind nicht nur für ein Plug-in gültig, sondern meist für eine ganze Plug-in Familie. So können Sie neben den Alphabeten für klassische Algorithmen auch die Kryptoanbieter in JCrypTool beeinflussen.
