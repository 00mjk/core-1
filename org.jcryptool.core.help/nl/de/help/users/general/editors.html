Editoren

JCrypTool enthält in der Standardinstallation zwei verschiedene Editoren: Text und Hex. Während der Texteditor nur für normale textbasierte Dateien geeignet ist können im Hexeditor beliebige Dateien geöffnet werden.

Beim Ausführen einer kryptografischen Operation wird das Ergebnis normalerweise im Editor desselben Typs geöffnet. D.h. haben Sie eine kryptografische Operation im Texteditor aufgerufen wird das Ergebnis normalerweise auch im Texteditor angezeigt (für den Hexeditor gilt entsprechendes). Je nach kryptografischer Operation kann es aber, vor allem um Darstellungsfehler auszuschließen, erforderlich sein, diese im Hexeditor zu öffnen.

Nach dem Öffnen einer Datei in einem der Editoren können Sie über das Bearbeiten -> Öffnen mit Menü diese im anderen Editor öffnen. Evtl. kann es dabei aber zu Darstellungsfehlern kommen.
