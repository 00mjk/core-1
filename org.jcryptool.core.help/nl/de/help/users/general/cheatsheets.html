Spickzettel

Spickzettel sind kleine, meist interaktive, Schritt für Schritt Anleitungen für JCrypTool. Diese müssen wie alles in JCrypTool von (Krypto) Plug-ins beigesteuert werden. Es gibt auch einige Basis Spickzettel die Ihnen die ersten Schritte in JCrypTool erleichtern.

Für welchen Spickzettel Sie sich auch entscheiden, aufrufen können Sie ihn über das Menü Hilfe -> Spickzettel.... Im Dialog können Sie den gewünschten Spickzettel auswählen und mit OK bestätigen. Es öffnet sich eine neue Sicht in der die einzelnen Schritte des Spickzettels angezeigt werden.

Im Spickzettel können Sie über die ? Icons die Hilfe mit tiefergehenden Informationen aufrufen. Über die Start und Erledigt Icons arbeiten Sie sich Stück für Stück durch die Todo-Liste. Beachten Sie, dass alle Icons bzw. die dahinterstehenden Operationen optional sind.
