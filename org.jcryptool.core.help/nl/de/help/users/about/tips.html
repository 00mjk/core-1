Maximimieren des Fensters eines Plugins

Falls Sie in JCT mehrere Fenster geöffnet haben, z.B. Editoren in der Standardansicht, die Kontexthilfe und einige Visualisierungen, und dann eines der Fenster maximieren wollen, doppelklicken Sie einfach den entsprechenden Reiter (bspw. auf den Reiter "Häufigkeits-Analyse" innerhalb der Visualisierungen).

JCrypTool im Netzwerk oder auf einem USB-Stick verwenden

Prinzipiell kann JCrypTool auch im Netzwerk oder auf einem externen Speichermedium wie einem USB-Stick oder einer USB-Festplatte verwendet werden. Dazu müssen Sie jedoch eine Änderung in einer Konfigurationsdatei vornehmen. Konkret müssen Sie den Pfad zum Workspace-Ordner anpassen. In diesem Ordner speichern alle Plug-ins die von Ihnen getroffenen Einstellungen.

Windows/ Linux:
Öffnen Sie die JCrypTool.ini in Ihrem Installationsordner und suchen Sie nach der Zeile -data @user.home/documents/.jcryptool. Hier müssen Sie @user.home/documents/.jcryptool durch einen in Ihrer Umgebung verfügbaren absoluten Pfad ersetzen.

macOS:
Die durchzuführende Änderung ist identisch wie unter Windows oder Linux. Allerdings befindet sich die JCrypTool.ini auf macOS unter JCrypTool.app\Contents\MacOS in Ihrem Installationsordner.
