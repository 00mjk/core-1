Über JCrypTool

Release Candidate 5 - August 2011

JCrypTool ist eine moderne und umfangreich erweiterbare E-Learning Platform rund um die Themen Kryptografie, Kryptoanalyse und IT-Security. Es unterstützt Lernende und Lehrende, Entwickler und andere an der Kryptografie Interessierte in der Anwendung und bei der Analyse von kryptografischen Mechanismen mit einer modernen und einfach zu verwendenden Plattform. JCrypTool wird als plattformunabhängige Open Source Software zur Verfügung gestellt und ist Teil des CrypTool Open Source Projekts.

JCrypTool ist unter der Eclipse Public License (EPL) Version 1.0 lizensiert.

Mehr Informationen und Unterstützung zur Kryptografie E-Learning Plattform finden Sie unter http://jcryptool.sourceforge.net. Dort finden Sie auf eine Präsentation für Ihre ersten Schritte mit JCrypTool (sowohl als Anwender als auch als Entwickler).

Der Source Code der Plug-ins, die zum JCrypTool Core gehören, ist unter http://sourceforge.net/projects/jcryptool verfügbar. Den Source Code zu den meisten restlichen JCrypTool Plug-ins finden Sie unter http://sourceforge.net/projects/jctplugins.
