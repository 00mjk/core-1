ElGamal

Das Elgamal-Verschlüsselungsverfahren ist ein asymmetrisches kryptographisches Verfahren, das auf der Idee des Diffie-Hellman-Schlüsselaustauschs aufbaut. Es kann zum Verschlüsseln und in einer Variante auch zum digitalen Signieren verwendet werden.

ElGamal finden Sie ausführlich erklärt unter den Visualisierungen in JCT: Visualisierungen → ElGamal-Kryptosystem
