Datei Explorer

Die Datei Explorer Sicht bietet einen einfachen Zugriff auf das Dateisystem.
Sie können Dateien öffnen in dem Sie auf den Dateinamen doppelklicken. Die gewählte Datei wird anschließend im Hexeditor geöffnet. Über das Öffnen mit Kontextmenü können Sie eine Datei in einem beliebigen JCrypTool Editor öffnen.

Zusätzlich können Sie über das Kontextmenü in der Sicht verschiedene Funktionen für Dateien und Verzeichnisse aufrufen:

  • Umbenennen
  • Kopieren
  • Einfügen
  • Löschen
  • Entschlüsseln/Verschlüsseln

Achtung: Dateioperationen wie Umbenennen und Löschen können nicht rückgängig gemacht werden! Insbesondere beim Löschen von Dateien müssen Sie beachten, dass diese nicht in den Papierkorb verschoben werden, sondern direkt und unwiderruflich nach dem Bestätigen der Sicherheitsabfrage gelöscht werden.

Verschlüsselung/Entschlüsselung

Das Menü Verschlüsselung/Entschlüsselung enthält dieselben Einträge wie die entsprechenden Menüpunkte in der Menüleiste und die Krypto Explorer Sicht.

Das Kontextmenü des Datei Explorers kann von anderen Plug-ins um eigene Einträge erweitert werden.
