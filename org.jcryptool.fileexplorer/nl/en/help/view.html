File Explorer

The File Explorer view offers easy file system access.
You can open a file by double clicking on the name which then opens the hex editor. The selected file will be opened in the Hexeditor. You can use the context menu entry Open with to open the selected file in an arbitrary JCrypTool editor.

There are additional functions available in the context menu:

  • Rename
  • Copy
  • Paste
  • Delete
  • Algorithms

Attention: It is not possible to undo file operations like rename or delete. Deleted files are not moved into the trash. Instead they will be deleted directly, and this operation cannot be undone.

Algorithms

The Algorithms menu contains the same algorithms as in the top menu.

The File Explorers' context menu can be extended by other plug-ins with new entries.
